# Stripe NavBar

Stripe NavBar is a clone of NavBar from [stripe](https://stripe.com)

## Installation

Use the package manager [npm](https://www.npmjs.com/) to [yarn](https://yarnpkg.com/) install.

```bash
yarn install
yarn start
```
